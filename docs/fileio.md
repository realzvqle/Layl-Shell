# File Operations

- fcreate [file]
    - creates a file
- fdelete [file]
    - deletes a file
- fcopy [file] [file2]
    - copys file to file2
- fread [file]
    - reads a file and prints its contents
- fwrite [file] [content]
    - writes content into file
- fprint [file] [content]
    - writes content into file without erasing the content inside
